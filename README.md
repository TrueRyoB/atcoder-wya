# AtCoder Where You Are

少ない問題数で、ユーザーの「真の実力帯」を正確に測定・可視化するフロントエンド完結型のツールです。

## 概要

AtCoderのレーティングは「コンテストで何問解けたか・どれだけ早く解けたか」で決まりますが、本ツールはそれとは異なる指標で実力を測ります。**特定の難易度の問題を、その難易度に見合った制限時間内に解けるか**を基準に、あなたの「真の適正難易度帯」を特定します。

## 対象ユーザー

- **伸び悩みを感じている方**: レートが停滞していても、真の実力は着実に伸びているかもしれません。週次測定の履歴から成長を可視化します。
- **競プロから復帰した方**: ブランク後のレートは過去の実績。現在の実力に合った練習難易度を素早く特定します。

## 特徴

- **フロントエンド完結**: インストール不要、アカウント登録不要。ブラウザだけで動作します。
- **短時間で高精度**: 複数の独立した推定手法を組み合わせることで、最小限の問題数から適正難易度を特定します。
- **成長の記録**: 測定履歴をローカルに保存し、折れグラフで成長を可視化します。
