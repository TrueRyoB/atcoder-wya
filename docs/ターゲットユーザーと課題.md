# ターゲットユーザーと課題

## ユーザーA: 伸び悩み勢

**課題**: レーティングが停滞しており、自分が成長しているのか実感できない。

**背景**: AtCoderのレートはコンテストのパフォーマンスに依存するため、実力が伸びていてもコンテストの出来次第でレートが上下する。地道な精進の効果が見えにくい。

**本ツールが提供する価値**: 週次測定の履歴を折れグラフで示すことで、レートに依存しない成長の軌跡を可視化する。

## ユーザーB: 復帰勢

**課題**: 競プロから離れていた期間があり、現在のレートが実力より高く（過剰評価）、適切な練習難易度がわからない。

**背景**: AtCoderのレートは下がりにくい設計のため、復帰時に過去のレートが現実と乖離したまま残る。難易度を誤ると、解けない問題に時間を浪費してしまう。

**本ツールが提供する価値**: 1回の測定で現在の適正難易度帯を特定し、適切なトレーニングメニューへ誘導する。

## 共通の前提

両ユーザーともPC環境で競プロを行っており、ブラウザアクセスに障壁はない。
