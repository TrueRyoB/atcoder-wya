# ロードマップ

本ツールを単なる「一過性の測定器」に留めず、ユーザーの継続的な成長を支援する「精進の伴走プラットフォーム」へと進化させるため、以下の機能をロードマップとして見据えています。

## 将来機能

### パーソナライズ問題集の自動生成

測定された適正難易度に基づき、AtCoderの問題プールから「ユーザーがまだACしていない、かつ今解くべき最適な問題」をセレクトしたオリジナル問題集をワンクリックで生成する機能。

### データ蓄積による二分探索のさらなる高速化

過去の測定ログ（正誤の傾向）をローカルに蓄積・解析することで、次回以降の二分探索の初期値やステップ幅をさらに最適化し、測定にかかる手間を極限まで削減する。
